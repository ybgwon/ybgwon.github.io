Posted on 2019-11-12 Disqus:

disqus 추가하기   hexo disqus next

<2019-11-08 금 01:09> hexo next에 disquss 를 추가하는 방법이다. 요약하자면

  1. disqus 가입

  2. themes/next 디렉토리의 _config.yml 파일 수정 comments: 섹션에 있다. 

    # Disqus
disqus:
  enable: true
  shortname: ybgwon
  count: true
  lazyload: false
  #post_meta_order: 0

  3. deploy 

    $ hexo g -d

관련 링크

https://goodgoodjm.github.io/blog/disqus-with-hexo-next/ https://theme-next.org/docs/third-party-services/comments
